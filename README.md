## 📚 Sobre o Projeto

Este repositório contém implementações práticas das principais **estruturas de dados em Java**, com foco em aprendizado, prática e preparação para entrevistas técnicas.

---
## 🎯 Objetivos

- 📌 Aprender estruturas de dados na prática  
- 🧠 Melhorar lógica de programação  
- 💼 Preparar para entrevistas e concursos  

---

## 🧩 Estruturas Implementadas

### 🔹 Listas
- `ArrayList`
- `LinkedList`

📌 Estruturas dinâmicas para armazenar coleções de dados

---

### 🔹 Pilha (Stack)
- `Stack`

📌 LIFO (Last In, First Out)  
📌 Último que entra, primeiro que sai

---

### 🔹 Fila (Queue)
- `Queue`
- `PriorityQueue`

📌 FIFO (First In, First Out)
📌 Fila com prioridade organiza de acordo com maior prioridade

---

### 🔹 Mapas (Map)
- `HashMap`
- `Hashtable`
- `TreeMap`

  
📌 Estrutura chave → valor


| Estrutura | Característica          |
| --------- | ----------------------- |
| HashMap   | 🔥 Rápido, não ordenado |
| Hashtable | 🔒 Sincronizado         |
| TreeMap   | 📊 Ordenado             |
